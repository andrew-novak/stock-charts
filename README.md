# stock-charts

Desktop app for visualizing stock charts.

The app can be downloaded [here](https://andrewnovak.co.uk/project/7).

### To run in development mode:

1. Install dependencies:

```
npm i
```

2. Start:

```
npm run start
```

### To build for production:

1. Install dependencies:

```
npm i
```

2. Build:

```
npm run release
```
